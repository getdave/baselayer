Baselayer Component Documentation

Baselayer is a small set of CSS/JS extensions built on top of Zurb's Foundation 5 framework. The patterns below describe the additional components that are made available.

Grid

Baselayer makes use of Foundation 5's mobile-first grid. It also extend this however, to include an inline-block grid type. Simply apply the .row--ib modifier to the standard Foundation .row element and your nested grid items will switch to using inline block affording the ability to vertically align.
